### Understanding the AZ-900 Exam

**What is AZ-900?**

The Microsoft Azure Fundamentals (AZ-900) certification proves you understand:
- Basic cloud concepts
- Core Azure services
- Azure pricing and support
- Security, compliance, and governance

**Who is it for?**
- Complete beginners to cloud
- Business professionals wanting to understand Azure
- Technical people switching to Azure
- Anyone wanting to start an Azure career

#### Strategy 2: Active Recall with Flashcards

Make flashcards for key facts:

```
Examples:

FRONT: What does IaaS stand for?
BACK: Infrastructure as a Service

FRONT: What's the Azure SLA for VMs with 2+ instances in availability set?
BACK: 99.95%

FRONT: What's the shared responsibility in SaaS?
BACK: Provider: Everything except data & access. Customer: Data & access control

FRONT: What tool estimates Azure costs before deployment?
BACK: Azure Pricing Calculator

FRONT: What's the difference between Azure Policy and RBAC?
BACK: Policy = What resources can be created. RBAC = Who can do what

Use apps like: Anki, Quizlet, or physical cards
Review daily!
```

### Common Tricky Questions

**Watch out for these question types:**

```
┌──────────────────────────────────────────────────────────────┐
│ TRICKY QUESTION PATTERNS                                     │
│                                                              │
│ PATTERN 1: "ALWAYS" / "NEVER" Questions                      │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ Example: "Data in Azure Storage is ALWAYS encrypted"     │ │
│ │                                                          │ │
│ │ Answer: TRUE                                             │ │
│ │ Why: Automatic encryption, can't be disabled             │ │
│ │                                                          │ │
│ │ Tip: If statement says "always" about a security         │ │
│ │ feature in Azure, often TRUE                             │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                                              │
│ PATTERN 2: "Which provides the MOST control?"               │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ Options: IaaS, PaaS, SaaS                                │ │
│ │                                                          │ │
│ │ Answer: IaaS (always)                                    │ │
│ │ Why: Most control = IaaS                                 │ │
│ │      Most convenience = SaaS                             │ │
│ │      Middle ground = PaaS                                │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                                              │
│ PATTERN 3: "Best practice to reduce costs?"                 │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ Look for:                                                │ │
│ │ • Reserved instances                                     │ │
│ │ • Auto-shutdown                                          │ │
│ │ • Right-sizing VMs                                       │ │
│ │ • Deleting unused resources                              │ │
│ │                                                          │ │
│ │ NOT: "Use more VMs" (obviously wrong)                    │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                                              │
│ PATTERN 4: "Which allows you to [manage multiple subs]?"    │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ Options: Resource Groups, Subscriptions, Management      │ │
│ │ Groups                                                   │ │
│ │                                                          │ │
│ │ Answer: Management Groups                                │ │
│ │ Why: Only MGs can contain multiple subscriptions         │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                                              │
│ PATTERN 5: "What stores Azure Monitor logs?"                │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ Common wrong answers:                                    │ │
│ │ • Azure Storage                                          │ │
│ │ • Azure SQL Database                                     │ │
│ │                                                          │ │
│ │ Correct: Log Analytics Workspace                         │ │
│ │ (This specific answer is tested frequently!)             │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 🎯 Final Practice Questions

**Question 1:**
You need the MOST control over operating system. Which service model?

A) SaaS  
B) PaaS  
C) IaaS  
D) All provide same control

**Question 2:**
What stores Azure Monitor logs?

A) Azure Storage Account  
B) Log Analytics Workspace  
C) Azure SQL Database  
D) Resource Group

**Question 3:**
Single VM with Premium SSD has what SLA?

A) 99.9%  
B) 99.95%  
C) 99.99%  
D) No SLA

**Question 4:**
Which lets you manage multiple subscriptions?

A) Resource Groups  
B) Management Groups  
C) Azure Policy  
D) RBAC

**Question 5:**
Data in Azure Storage is ALWAYS encrypted?

A) True  
B) False

---

### ✍️ Answers

**1: C** - IaaS gives most control (you manage OS)  
**2: B** - Log Analytics Workspace (not Storage!)  
**3: A** - 99.9% (single VM with Premium SSD)  
**4: B** - Management Groups contain subscriptions  
**5: A** - True (automatic, can't disable)
